HTML


Head Element

  • The head element contains information about the webpage.
  • The head element is also where we link our external css and javascript files.

Body Element

  • We use semantic tags to tell the browser what it is we are displaying.
  • Inside the opening semantic tag we give class and id attributes to target specific elements for styling and programatic code in our CSS and Javascript respectively.
  • When we add any media i.e image, audio or movie files, we must also use the alt attribute to tell any screen reading software what we are displaying.
  • This alt attribute is also displayed if the browser cannot display the file.

CSS

  • CSS file are linked in the head element of a HTML document. the reason for this location is so the browser downloads the file before the HTML body and applies any styles as they load.
  • In CSS we use both the class and id to target syling and animations. in this webpage for example the class targets all four note sections and the id targets individual section.
  • By doing this we can apply global styling to all cards then specific styling to differentiate the different subject.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • We can declare colours in different ways stated a color name, using hexedecimal denoted by '#', and we can use rgb, rgba, dsa and a few others check out css documentation.
  • We can specify size in alot of ways now but in this example page we have used '%' for page width and pixels 'px' for height and font size
